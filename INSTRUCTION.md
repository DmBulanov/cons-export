# Cons Export — инструкция для юристов

Инструмент помогает **найти практику в онлайн-КонсультантПлюс** и **сохранить документы на компьютер** без ручного «Сохранить» по одному файлу.

Работает только с **браузерной** версией КонсультантПлюс (вход через сайт). Локальная программа `cons.exe` не нужна.

---

## Что понадобится один раз

1. **Доступ** к онлайн-КонсультантПлюс (логин и пароль от сервисного центра).
2. Браузер **Google Chrome** (или Edge на базе Chromium).
3. Установленное расширение **Cons Export** (см. ниже).

---

## Установка расширения (5 минут)

1. Получите папку `extension` из этого проекта (скачайте репозиторий с GitHub и распакуйте, если нужно).
2. Откройте в Chrome адрес: `chrome://extensions`
3. Включите переключатель **«Режим разработчика»** (обычно справа вверху).
4. Нажмите **«Загрузить распакованное расширение»**.
5. Укажите папку **`extension`** (внутри неё должен быть файл `manifest.json`).
6. Закрепите иконку Cons Export на панели Chrome (пазл → булавка).

Готово. Python, Cursor и другие программы для работы инструмента не нужны.

---

## Настройка сохранений без лишних окон (рекомендуется)

Чтобы Chrome не спрашивал «куда сохранить» на каждом файле:

1. Откройте `chrome://settings/downloads`  
   (в расширении есть кнопка «Открыть настройки загрузок»).
2. Укажите удобную папку загрузок (например «Документы» или «Загрузки»).
3. **Выключите** пункт «Всегда указывать место для скачивания».

В Cons Export в поле **«Папка внутри Загрузки»** укажите, например, `ConsExport` или `Практика`.  
Файлы появятся в:  
`ваша папка загрузок Chrome` → `ConsExport` (или как назвали).

## Как работать каждый день

### 1. Войдите в КонсультантПлюс

Откройте [login.consultant.ru](https://login.consultant.ru/), введите логин и пароль.  
Должна открыться система на адресе вида `online.consultant.ru`.

### 2. Откройте Cons Export

Нажмите иконку расширения на панели Chrome.

### 3. Опишите, что искать

В поле **«Какую практику найти?»** напишите обычным языком, например:

> взыскание задолженности по аренде нежилого помещения арбитраж

### 4. Выберите область

- **Судебная практика** — предпочтительно для подборки судебной практики;
- **Всё по запросу** — если нужны ещё формы, готовые решения и т.п.

### 5. Формат файла

По умолчанию выбран **DOCX** (удобно для Word и Legal IDE).  
При необходимости смените на PDF, RTF, TXT или HTML.

- **DOCX / PDF / RTF** — сохранение через меню КонсультантПлюс (как при ручном «Сохранить в файл»).
- **TXT / HTML** — быстрее, без диалогов КП; текст снимается со страницы документа.

### 6. Запуск

| Кнопка | Что делает |
|--------|------------|
| **Найти** | Только показывает список найденных документов |
| **Найти и сохранить** | Ищет и сразу сохраняет найденное в выбранном формате |
| **Экспорт текущего списка** | Сохраняет уже отсканированный список на экране |
| **Сохранить текущий** | Сохраняет один открытый документ |
| **Стоп** | Останавливает массовое сохранение |

Дождитесь окончания: в блоке «Прогресс» будет видно, сколько файлов обработано.

### 7. Где лежат файлы

Папка загрузок Chrome → подпапка из настроек расширения (по умолчанию `ConsExport`).

Дальше файлы можно перенести в папку судебного дела или открыть в Legal IDE.

## Типовые ситуации

**Расширение «не видит» страницу**  
Убедитесь, что активна вкладка `online.consultant.ru` и вы вошли в систему. Обновите расширение на `chrome://extensions` (кнопка обновления).

**Список пустой после «Найти»**  
Уточните формулировку или выберите «Всё по запросу». Проверьте, что поиск в самом КП отработал (на вкладке должна появиться выдача).

**Снова появляется окно «Сохранить как»**  
В настройках загрузок Chrome выключите «Всегда указывать место для скачивания». Для массовой работы без окон можно временно выбрать TXT.

**Нужны не все документы из выдачи**  
Сначала нажмите **Найти**, посмотрите список в логе расширения, затем сохраняйте выборочно через открытие нужных документов и **Сохранить текущий**, либо уточните запрос и повторите поиск.

**Сессия КонсультантПлюс завершилась**  
Войдите снова через login.consultant.ru и повторите действие.

## Важно о доступе и лицензии

- Используйте только **свой** легальный доступ к КонсультантПлюс.
- Инструмент повторяет действия, которые вы можете выполнить вручную в интерфейсе (поиск и сохранение).
- Соблюдайте пользовательское соглашение и правила вашего сервисного центра.
- Не передавайте логин и пароль третьим лицам и не публикуйте выгрузки с ограниченным доступом вне рамок вашей работы.

## Краткая шпаргалка

1. Войти в онлайн-КП.  
2. Cons Export → ввести тему практики.  
3. Область: судебная практика.  
4. Формат: **DOCX** (или другой).  
5. **Найти и сохранить**.  
6. Забрать файлы из папки загрузок / `ConsExport`.
