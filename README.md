# Cons Export

Расширение Chrome для поиска и массового сохранения документов из **браузерной** версии КонсультантПлюс (`online.consultant.ru`).

**Инструкция для юристов:** [INSTRUCTION.md](INSTRUCTION.md)

## Возможности

- ввести запрос прямо в расширении («какую практику найти»);
- отфильтровать **судебную практику** или искать по всему;
- сохранить найденные документы (по умолчанию **DOCX**, также TXT / HTML / PDF / RTF);
- складывать файлы в подпапку внутри «Загрузки» Chrome без лишних окон (после одноразовой настройки).

## Установка

1. Скачайте или клонируйте репозиторий.
2. Chrome → `chrome://extensions` → включите **Режим разработчика**.
3. **Загрузить распакованное расширение** → папка `extension`.
4. Закрепите иконку Cons Export.

Подробный пошаговый сценарий для ежедневной работы — в [INSTRUCTION.md](INSTRUCTION.md).

## Разработка

Исходники расширения в `extension/`. После правок нажмите **Обновить** на карточке расширения в `chrome://extensions`.
